                            💡 About MzansiLM:

                            MzansiLM is a 125M-parameter base model on Hugging Face. It is NOT a chat API — it's a research foundation model trained on all 11 SA languages. For chat, use DeepSeek or Hugging Face's free inference API.
                        

                        

                            🏠 Run MzansiLM Locally:


                            pip install transformers torch


                            from transformers import AutoModelForCausalLM, AutoTokenizer

                            model = AutoModelForCausalLM.from_pretrained("anrilombard/mzansilm-125m")

                            tokenizer = AutoTokenizer.from_pretrained("anrilombard/mzansilm-125m")


                            GitHub: Anri-Lombard/sallm
